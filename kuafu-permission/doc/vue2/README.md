# webapp-box - Vue 2.7 + Vite 项目

## 项目说明

本项目使用 Vue 2.7 + Vite 5.x 构建。

## 环境要求

- Node.js >= 16.x (推荐使用 v18.x)
- npm >= 8.x

## 安装依赖

```bash
npm install
```

## 开发

启动开发服务器：

```bash
npm run dev
# 或
npm start
```

## 构建

### 开发环境构建
```bash
npm run build:dev
```

### QA环境构建
```bash
npm run build:qa
```

### 生产环境构建
```bash
npm run build:prd
```

## 代码检查

```bash
npm run lint
```
